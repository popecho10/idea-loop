## Verdicts

### IDEA-20260714-02 · Self-observation system — **START**

Prior art exists (mood trackers, quantified-self apps) but all of it optimizes for *capture*, not *review* — the insight loop is missing, which is exactly what an AI agent can add. Effort for a minimal version is low: a one-line daily log plus a monthly analysis prompt.

**Minimal experiment (4 weeks):** log 3 fields daily (energy 1–5, one decision made, one sentence of context) in a single markdown file; at week 4, run an analysis prompt asking for patterns and one falsifiable prediction about week 5. **Kill criterion:** if logging takes >2 minutes/day or gets skipped >8 of 28 days, the format is wrong — stop and redesign rather than push through.

See `examples/self-observation/` for the full experiment plan.

### IDEA-20260712-01 · Era-feel observations — **EXPLORE**

### IDEA-20260711-01 · API-quota resale side-business — **SHELVE**

Web research: the resale market for AI-API quota is saturated by grey-market operators selling at 10–30% of official prices, largely by violating provider terms of service; providers are actively banning the practice. A compliant reseller cannot compete on price, and the compliance risk is asymmetric. **Re-activation condition:** revisit only if providers introduce an official reseller/affiliate program with sanctioned margins.
